## 📖 Sobre o Projeto

O **Tecteca Mobile** é um aplicativo cross-platform desenvolvido com [React Native](https://reactnative.dev/). Nossa missão é oferecer uma experiência de leitura imersiva e social. Com suporte a **múltiplos perfis**, o app permite que toda a família desfrute de centenas de livros, interaja com amigos e personalize seus próprios avatares de forma muito divertida!

Para facilitar a navegação pelo nosso código e processos operacionais, dividimos a documentação da seguinte forma:

- 🏗️ **[Estrutura e Arquitetura](./ARCHITECTURE.md):** Um mergulho na organização da nossa codebase (pastas `src/`), escolhas arquiteturais e bibliotecas base, como Zustand e TanStack Query.

## 🛠️ Tecnologias Principais

O projeto faz uso das tecnologias mais modernas do ecossistema mobile atual para garantir fluidez e escalabilidade:

| Categoria | Tecnologia & Stack |
| :--- | :--- |
| **Framework Base** | ⚛️ React Native (`v0.81.1`) + TypeScript |
| **Roteamento** | 🛣️ React Navigation (`v7`) |
| **Estado e Cache** | 🐻 Zustand + ⏳ TanStack React Query (`v5`) |
| **Estilização** | 💅 NativeWind (TailwindCSS) + Styled Components |
| **Storage Rápido** | 💾 React Native MMKV |
| **Integração de Rede** | 🌐 Axios (REST) |
| **Formulários** | 📝 React Hook Form + Yup |
| **UI & UX** | 🎭 React Native Reanimated + Moti (Animações fluidas) |
